歯笛は、普通の口笛より「歯で音を出している」と感じやすいですが、私の見立てでは、歯は主役というより**安定した細い開口部・エッジ・ノズル**です。音程を大きく決めているのは、その奥にある舌・口腔・喉側の共鳴腔だと思います。

口笛研究では、基本周波数が声道の共振周波数に対応すること、また舌を奥に引いて共鳴腔を広げると低く、舌を上あご側に近づけて狭くすると高くなる、という説明がされています。ただし、口笛は完全にヘルムホルツ共鳴だけで説明しきれるわけではなく、息の速度によるオーバーブロー的な変化もあるようです。歯笛は分類上も「唇の狭め」ではなく「口腔内の狭め」による口笛として扱われていて、この点はかなり示唆的です。

なので、練習の中心は「強く吹く」ではなく、**歯の隙間を一定にしたまま、舌と息だけで共鳴条件を再現する能力を上げること**になると思います。

最初にやる価値があるのは、ロングトーンです。いちばん安定して鳴る音を選び、できるだけ弱い息で音が立ち上がる点を探します。そこで5〜10秒、音程を揺らさず維持します。目的は音量ではなく、発音の閾値を下げることです。小さい息で鳴る状態ほど、歯・舌・口腔の配置が合っている可能性が高いです。スマホのチューナーかスペクトラム表示アプリで、音程の揺れとノイズ成分を見るとよいです。口笛音は純音に近いが、微細な揺らぎを含むと報告されています。

次に、「母音ポジション」を使って舌の地図を作るのがよさそうです。研究では、日本語5母音の声道形状を使った模型実験で、特に /e/ の形がよい結果を出したとされています。歯笛でも、まず「エ」に近い舌位置を基準にして、そこから「イ」寄りにすると高音、「ウ・オ」寄りにすると低音、というふうに探ると整理しやすいはずです。

練習としては、同じ息の強さで「エ → イ → エ → ウ」のようにゆっくり動かし、音程がどう変わるかを記録します。ここで大事なのは、顎や歯の隙間をなるべく変えないことです。歯笛では開口部がもともと硬く固定されやすいので、変えるべき変数は舌です。顎・唇・頬まで一緒に動くと、何が効いたのかわからなくなります。

音域を広げるなら、低音と高音で練習方針を分けたほうがいいです。低音は口腔容積を広げ、息をやや遅くする。高音は舌を上げて共鳴腔を狭くし、息の速度を少し上げる。ただし高音で力任せに吹くと、ノイズだけ増えて共鳴が外れます。研究でも、低い音では遅め、高い音では速めの息が必要と説明されています。

最後に、上達のためには「音を出す練習」と「音楽的に使う練習」を分けるのがよいです。前者はロングトーン、最弱発音、舌位置の再現。後者は半音階、跳躍、ビブラート、アタックです。特に歯笛は発音点が鋭いので、タンギングに相当するものは「トゥ」よりも、無声音の「ス」「ツ」に近い微細な息の立ち上げが合うかもしれません。これは推論ですが、歯の隙間をノズルとして使うなら、舌先で強く遮断するより、息圧の立ち上げだけで開始したほうが音程が崩れにくいはずです。

まとめると、最適なアプローチはたぶんこれです。**歯の形で吹く技術ではなく、歯を固定ノズルとして使い、舌で共鳴腔を精密にチューニングする技術として練習する。** その意識に切り替えるだけで、練習の焦点はかなり明確になると思います。
