Dartで、ISO8601形式の日時文字列のリストを受け取り、日付(yyyy-MM-dd)ごとの件数をMap<String, int>で返す関数を書いてください。タイムゾーンは考慮せず、文字列の日付部分で数えて構いません。












文字列先頭の `yyyy-MM-dd` 部分だけをキーにして数えるなら、`substring(0, 10)` で十分です。

```dart
Map<String, int> countByDate(List<String> isoDateTimes) {
  final counts = <String, int>{};

  for (final value in isoDateTimes) {
    if (value.length < 10) {
      throw FormatException('Invalid ISO8601 datetime string: $value');
    }

    final date = value.substring(0, 10);
    counts[date] = (counts[date] ?? 0) + 1;
  }

  return counts;
}
```

使用例です。

```dart
void main() {
  final values = [
    '2026-07-05T10:00:00',
    '2026-07-05T23:30:00',
    '2026-07-06T00:15:00',
  ];

  final result = countByDate(values);

  print(result);
  // {2026-07-05: 2, 2026-07-06: 1}
}
```

タイムゾーンや `DateTime.parse()` を使わないので、`2026-07-05T23:00:00Z` のような文字列でも、単純に `2026-07-05` として数えます。
