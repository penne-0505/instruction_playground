# Instruction Playground

Instructionやpromptを管理・テスト・改善するためのリポジトリです。

## 概要
使用するinstructionや、試し途中のものを保管するためのリポジトリです。大まかに、ディレクトリ配下の構造を持ちます。
```
- general_instructions/  (一般用途のinstruction)
  - GENERAL/  (ベースとなるinstruction)
  - chatgpt/
  - claude/
  - gemini/
- meta_contents/  (意図やアーカイブ済みのもの)
  - idea_snipet/
  - intent/
  - old_or_current/
  - old_test_prompts/
  - meta_snipet.md  (Geminiのプロンプト適用時の改変防止用スニペット)
- playground/  (未完成品の保管・作業用ディレクトリ)
- test_prompts/  (モデルの傾向を見るためのリファレンス集)
- CURRENTRY_USING.md  (現在使用中のinstructionへのリンク)
```
